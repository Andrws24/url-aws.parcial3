# URL Shortener AWS - Parcial Sistemas Distribuidos

Proyecto desarrollado con AWS Lambda, API Gateway, DynamoDB y Terraform.

## Tecnologías

- AWS Lambda
- API Gateway
- DynamoDB
- Terraform
- Node.js
- TypeScript
- S3 + CloudFront

## Módulos

### Backend
- Acortamiento de URLs
- Estadísticas
- Redirección

### Frontend
- Formulario de acortamiento
- Visualización de estadísticas

## Infraestructura

Toda la infraestructura fue desplegada usando Terraform.
